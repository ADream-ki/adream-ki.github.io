# AI助手应用

这是一个基于大语言模型的智能对话助手应用，提供自然语言处理和智能回答功能。

## 主要特性

- 🤖 集成OpenAI GPT模型
- 💬 流式对话体验
- 📝 对话历史记录
- 🎨 优雅的聊天界面
- 📱 移动端适配

## 技术栈

- **前端**: Next.js 14 + TypeScript
- **AI服务**: OpenAI API
- **样式**: Tailwind CSS
- **状态管理**: React Hooks
