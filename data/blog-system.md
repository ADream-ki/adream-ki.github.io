# 个人博客系统

这是一个基于Next.js 14和Tailwind CSS构建的现代化个人博客系统，具有响应式设计和优雅的二次元风格界面。

## 主要特性

- 🎨 现代化UI设计，支持深色模式
- 📱 完全响应式布局
- 🔍 强大的搜索和筛选功能
- ⚡ 静态站点生成，加载速度快
- 🏷️ 标签和分类系统
- 📝 支持Markdown和MDX格式

## 技术栈

- **前端框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS + 自定义设计系统
- **语言**: TypeScript
- **部署**: GitHub Pages + GitHub Actions
